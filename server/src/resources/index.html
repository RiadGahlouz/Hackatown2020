API Reference:

Routes:

/[get]→this page
/restos[get]→list of restaurants
/menus/{resto_id}[get]→menu of restaurant with id {resto_id}
/order[post]→order from resto. Send a json in the body with structure "order"
/orders/{order_id}[get]→current status of order with order_id

Json structure

Order
[ { "resto_id": id of resto, "menu_id": id of menu item }, ... ]

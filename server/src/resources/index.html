API Reference:

Routes:

/[get]→this page
/restos[get]→list of restaurants
/menus[get]→all available menu items
/menus/{resto_id}[get]→menu of restaurant with id {resto_id}
/order[post]→order from resto. Send a json in the body with structure "order"
/orders[get]→get all orders, past and present
/set_pos[post]→update the position of the delivery guy, use json like {"lat": 3.45345, "lon": 34.567, "alt": 105.03, "accr": 53.03, "accv": 5.33}
/get_pos[get]→get the position of the delivery guy

Json structure

Order json
{ "menu_item_ids": ["2", "4", ... ] }
